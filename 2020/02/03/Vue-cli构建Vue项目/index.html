Vue-cli构建Vue项目

创建新的Vue项目
  1. 按回车确认项目名或者在光标处修改，项目名不能以大写字母开头，不能包含特殊字符。
  2. 项目描述 可以输入也可以直接回车确认
  3. 作者 可以输入也可以直接回车确认
  4. 项目创建方式 可以选择运行+编译或者运行
  5. 是否安装vue-router 可以添加也可以在之后再手动添加
  6. 是否添加ESlint 可以添加也可以不添加
  7. 设置单元测试 可以添加也可以不添加
  8. e2e测试 可以添加也可以不添加
  9. 是否需要运行npm或者yarn 可以运行也可以不运行

创建好之后cd到项目目录里

然后手动npm install

运行

使用Vue-cli创建Vue项目

打开终端 安装Vue-cli

查看版本

创建项目

  1. 选择编译工具
  2. router是否使用历史模式
  3. 想在哪里配置
  4. 是否保存为模版

启动项目

打开项目首页 大功告成！
http://localhost:8080/
